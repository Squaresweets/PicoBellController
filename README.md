# PicoBellController

A bastardized version of [this](https://github.com/CrazyRobMiles/PICO-blue-chords/tree/master) project aimed at letting you control Campana or Mobel by pressing the bootsel button on the pico w.

It just acts as a bluetooth keyboard that sends space, so it also works with Anki.

To use it just drag the .uf2 file onto the pico.
